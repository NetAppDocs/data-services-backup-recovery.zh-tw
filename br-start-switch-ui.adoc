== 切換到不同的工作負載

您可以在NetApp備份和還原 UI 中切換到不同的工作負載。

.步驟
. 從控制台左側導覽中，選擇*保護*>*備份和還原*。
. 從頁面右上角，選擇*切換工作負載*下拉清單。
. 選擇您想要切換到的工作負載。
+
頁面刷新並顯示所選的工作負載。
