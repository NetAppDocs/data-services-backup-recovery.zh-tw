= NetApp Backup and Recovery針對 KVM 工作負載的已知限制
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
此處列出了與此版本不相容或無法很好地相容的平台、裝置或功能。仔細閱讀這些限制。

NetApp Backup and Recovery中的 KVM 工作負載的私人預覽版不支援以下操作和設定：



== 不支援的操作

私人預覽版不支援以下操作：

* 克隆、掛載或卸載虛擬機
* 將虛擬機器還原到備用位置
* 保護儲存在 SAN 上的虛擬機
* 保護應用程式
* 建立使用者定義的備份（僅支援從NetApp Console啟動的備份）




== 不支援的配置

不支援以下配置：

* 基於角色的存取控制 (RBAC)
* 直接連接到 KVM 主機的磁碟
* 跨多個 NFS 掛載點或共享的磁碟
* RAW磁碟格式
* NetFS 以外的儲存池類型（僅支援 NetFS）




== 故障排除說明

使用 KVM 工作負載的私有預覽版搭配 NetApp Backup and Recovery 時，請注意下列事項。

* 如果受保護的虛擬機器中未執行 QEMU Guest Agent，則備份可能會失敗。作為解決方法，請確保 `qemu-guest-agent`守護程序在啟動備份之前已在虛擬機器上執行。
