= NetApp備份和還原身分識別和存取管理 (IAM) 角色

:allow-uri-read: 
:nofooter:

:linkattrs:

[role="lead"]
NetApp Backup and Recovery 採用身分識別和存取管理 (IAM) 來管理每個使用者對特定功能和操作的存取權限。
