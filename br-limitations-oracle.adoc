= Oracle 資料庫工作負載的 NetApp Backup and Recovery 已知限制
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
此處列出了與此版本不相容或無法很好地相容的平台、裝置或功能。仔細閱讀這些限制。

在 NetApp Backup and Recovery 的 Oracle Database 工作負載預覽版本中，不支援下列動作：

* 離線備份


Oracle Database 僅在 Oracle Database 工作負載的私人預覽版本中支援使用 NFS、SAN 或 ASM SAN 進行獨立部署。
