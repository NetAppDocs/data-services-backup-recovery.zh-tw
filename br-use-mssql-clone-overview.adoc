= 使用NetApp Backup and RecoveryMicrosoft SQL 伺服器工作負載概述
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
使用NetApp Backup and Recovery將 Microsoft SQL Server 應用程式資料複製到相同或不同的 VM，以用於開發、測試或保護目的。您可以從 Microsoft SQL Server 工作負載的即時快照或現有快照建立複製。

* 創建克隆
* 刷新克隆
* 分裂克隆
* 刪除克隆
